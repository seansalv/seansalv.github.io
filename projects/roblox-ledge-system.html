Ledge Grabbing & Climbing System (Roblox)

This system handles ledge detection, grabbing, and climbing using Roblox Lua. It supports different ledge heights, windows, and blockers while keeping movement smooth and responsive. It also includes a wall-climbing variant with a decay mechanic to limit how long the player can stay on a surface.

Wall Climbing with Decay

Shows the wall-climbing behavior with a decay mechanic that gradually reduces climb time. This prevents infinite wall-cling and makes the system feel more skill-based and tunable.

Ledge Vaulting Demo

Demonstrates vaulting over different ledge heights, including short vaults, full climbs, and handling of awkward edge positions where the player should not snap or jitter.

What’s Implemented

  • Ledge detection via forward and downward raycasts to find valid grab points.
  • Finite state machine for idle, grab, climb, vault, wall-climb, and drop states.
  • Configurable rules for windows, blockers, and invalid ledges.
  • Animation hooks and event points for syncing movement with character animations.
  • Decay-based timer to control how long players can cling to walls.

Tech & Concepts

Roblox Lua, raycasting, state machines, character movement manipulation, animation events, and parameterized tuning for gameplay feel (ledge distance, raycast length, decay rate, and climb speeds).
