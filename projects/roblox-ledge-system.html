Ledge Grabbing & Climbing System (Roblox)

This system handles ledge detection, grabbing, and climbing using Roblox Lua. It supports different ledge heights, windows, and blockers while keeping movement smooth and responsive.

Showcase Video

What’s Implemented

  • Ledge detection using raycasts
  • State handling for idle, grab, climb, and drop
  • Edge cases like windows and blockers
  • Animation hooks for smooth transitions
